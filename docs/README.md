## 🎨 デザインテーマ

**ピーナッツ（スヌーピー）風**
- クリーム色の紙の背景
- 手描きフォント（Patrick Hand, Permanent Marker）
- コミックパネル風枠線
- 温かみのあるアナログな雰囲気
