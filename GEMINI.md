# Gemini Project Instructions

このファイルは Gemini CLI エージェントに対するプロジェクト固有の指示を定義します。

## 言語設定
- 回答はすべて日本語で行ってください。

## プロジェクト概要
- **プロジェクト名:** Tomatic
- **技術スタック:** React, Vite, CSS Modules
- **目的:** シンプルで使いやすいポモドーロタイマーアプリ

## コーディング規約
- コンポーネントは `src/components/` に配置する。
- スタイルはコンポーネントごとの `.css` ファイルで管理する。
- ロジックは可能な限りカスタムフック（`src/hooks/`）に抽出する。
- 型定義が必要な場合は JSDoc を使用する。

## 開発プロセス
- 機能追加の際は、まず既存のコンポーネント構造を確認すること。
- テストが可能な場合は、実装と同時にテストコードの提案も行うこと。
